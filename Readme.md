#**FitTrack Lite**

##User Persona
A beginner fitness enthusiast who wants a simple way to track workouts, monitor calories burned, and stay consistent without using complex fitness apps.

###Problem Statement
Many beginners struggle to track workouts consistently and lack a simple, visual way to monitor their fitness progress.

####Purpose
A clean, easy-to-use dashboard that allows users to log workouts, view progress, and stay motivated.
